Azure Stadium Ticket Booking Infrastructure

This project demonstrates a secure and scalable cloud infrastructure built using Microsoft Azure services. The architecture includes Virtual Machines, Load Balancer, Storage Account, Recovery Services Vault, Microsoft Entra ID, RBAC, NSG Rules, SAS Tokens, and Azure Policy.

Security & Governance

Network Security

NSG rules were configured to allow communication between frontend and application virtual machines while blocking direct access to the database VM.

Identity Management

Microsoft Entra ID and RBAC were implemented to manage users, groups, and secure role assignments.

Storage Security

SAS Tokens were generated for secure and temporary access to Azure Storage resources.

Backup & Recovery

Recovery Services Vault protects critical resources using backup policies and disaster recovery configurations.
